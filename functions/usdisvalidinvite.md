---
description: Checks if given invite is valid. (Returns true or false)
---

# $isValidInvite

#### Usage

```javascript
$isValidInvite[discord.gg/] //After // is your invite Code
```

#### Example

```javascript
bot.command({
name: "isvalid", 
code: `
$isValidInvite[https://discord.gg/HMUfMXDQsV]` //Returns true
})
```
