# $setBotAvatar

This function set's the bot's avatar

```text
$setBotAvatar[url]
```

```javascript
bot.command({
name: "setbotavatar",
code: `Set the bots avatar! 
$setBotAvatar[https://cdn.discordapp.com/avatars/746401637329010779/a3dc97600375b95156a33d0fccbf2c95.webp]`
})
```
