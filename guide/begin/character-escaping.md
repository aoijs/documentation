# Character Escaping

#### Escaping characters:

> `[` =&gt; `#RIGHT#` 
>
> `]` =&gt; `#LEFT#` 
>
> `;` =&gt; `#SEMI#` 
>
> `$` =&gt; `#CHAR#`
>
> `:` =&gt; `#RIGHT_CLICK#` 
>
> `<` =&gt; `#LEFT_CLICK#` 
>
> `=` =&gt; `#EQUAL#` 
>
> `}` =&gt; `#LEFT_BRACKET#` 
>
> `{` =&gt; `#RIGHT_BRACKET#` 
>
> `:` =&gt; `#COLON#` 
>
> `,` => `#COMMA#`
#### more:

Backspace =&gt; `\b`   
Form Feed =&gt; `\f`   
New Line =&gt; `\n`  
Carriage Return =&gt; `\r`  
Horizontal Tabulator =&gt; `\t`  
Vertical Tabulator =&gt; `\v`  
Single quote =&gt; `\'`  
Double quote =&gt; `\"`  
Backslash =&gt; `\`

{% hint style="info" %}
These character escapes are useful for using special characters inside functions, e.g. in the text field of [$sendMessage](../../functions/usdsendmessage.md).
{% endhint %}
