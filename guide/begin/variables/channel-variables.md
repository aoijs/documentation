# Channel Variables

{% hint style="danger" %}
Make sure you've already created the variable
{% endhint %}

### Setting the channel variable

```text
$setChannelVar[variable;value;channelID (optional)]
```

### Retrieving the variable value

```text
$getChannelVar[variable;channelID (optional)]
```
