# $setGuildIcon

This function set's the current guilds icon

```text
$setGuildIcon[url]
```

```javascript
bot.command({
name: "setGuildIcon",
code: `Set the guild icon
$setGuildIcon[https://cdn.discordapp.com/avatars/746401637329010779/a3dc97600375b95156a33d0fccbf2c95.webp]`
})
```
