# $serverChannelExists

This function checks if a channel exists in the current server. Returns Boolean

```javascript
$serverChannelExists[channel ID]
```

```javascript
bot.command({
name: "serverChannelExists",
code: `Does Channel Exist: $serverChannelExists[773361206496329728]`
})
```
