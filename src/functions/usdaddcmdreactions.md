# $addCmdReactions

`$addCmdReactions` adds single or multiple reactions to author's message.

### Usage

```php
$addCmdReactions[emoji;emoji;...]
```

### Field

| Field | Description | Type |
| :--- | :--- | :--- |
| emoji | A default Discord emoji or a custom emoji with ID | string |
