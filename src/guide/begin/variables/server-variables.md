# Server Variables

## Setting the value of the Server.

{% hint style="danger" %}
Make sure, you have the variables added already.
{% endhint %}

```php
$setServerVar[Variable;Value;Guild ID (Optional)]
```

## Retrieving value of the Server.

{% hint style="danger" %}
Make sure, you have the variables added already.
{% endhint %}

```php
$getServerVar[Variable;Guild ID (Optional)]
```
